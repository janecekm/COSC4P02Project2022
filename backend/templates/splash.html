Welcome to ChatBot

Brock University ChatBot

This ChatBot helps you by answering Brock University related questions, such as where is a course, when is a course, information regarding locations on campus, general information about where to find resources regarding programs and so on.

Canada Games ChatBot

This ChatBot answers questions regarding Niagara 2022 Canada Summer Games. Some of the questions it can answer include: where you can find tickets, what is the schedule for the games, when/where a particular sport is occuring, and where you can sign up for volunteering.
This project was done as part of the COSC 4P02 Course at Brock University. If you encounter any issues or discrepancies with the information available, please raise an issue in our GitHub repo here.
